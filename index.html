Master flex-direction, justify-content, and align-items

📋 Instructions

Your Mission: Use the dropdown controls to match the target layout shown in each challenge. You'll practice:

  • flex-direction: Controls the main axis (row, column, row-reverse, column-reverse)
  • justify-content: Aligns items along the main axis
  • align-items: Aligns items along the cross axis

💡 Watch how the flex items move as you change the properties!
